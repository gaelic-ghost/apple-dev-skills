# Skill Installation Guidance

## Apple skill bundle

- Repo: `https://github.com/gaelic-ghost/apple-dev-skills`
- Install (Codex, repo-local):
  - `npx skills add gaelic-ghost/apple-dev-skills -a codex`
- Install (Codex, global):
  - `npx skills add gaelic-ghost/apple-dev-skills -a codex -g`

## Broader discovery

- Install discovery bundle:
  - `npx skills add vercel-labs/agent-skills`
- Find skills:
  - `npx skills find <query>`

If user wants help, offer step-by-step guided install.
